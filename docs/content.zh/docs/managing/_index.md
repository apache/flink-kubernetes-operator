# Managing Flink Jobs

This section covers operating the jobs the operator runs: their lifecycle from start to deletion, upgrades and recovery under [Job Management]({{< ref "docs/managing/job-management" >}}), savepoints and checkpoints under [Snapshot Management]({{< ref "docs/managing/snapshot-management" >}}), zero-downtime rollouts under [Blue/Green Deployments]({{< ref "docs/managing/bluegreen-deployments" >}}), and automatic right-sizing under [Autoscaler]({{< ref "docs/managing/autoscaler" >}}) and [Autotuning]({{< ref "docs/managing/autotuning" >}}).
